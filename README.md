# 🏠 California Housing Price Prediction

Machine Learning model to predict California house prices.

## 🏆 Results
| Model | R² Score |
|-------|----------|
| Gradient Boosting ⭐ | 0.9613 |
| Random Forest | 0.9604 |
| Ridge Regression | 0.9176 |
| Linear Regression | 0.9176 |

**Best Accuracy: 96.1% | Avg Error: $18,873 per house**

## 🛠️ Libraries Used
- Python, Scikit-Learn, Pandas, NumPy
- Matplotlib, Seaborn

## 📊 Features Used
- Median Income, House Age, Rooms, Bedrooms
- Population, Location (Latitude/Longitude)
- RoomsPerPerson, BedroomRatio, PopulationDensity

## 🚀 How to Run
pip install scikit-learn pandas numpy matplotlib seaborn
python california_housing_prediction.py
